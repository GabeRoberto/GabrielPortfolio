# Personal Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Vite.

## Features

- Responsive design
- Dark/Light mode
- Interactive UI elements
- Project showcase
- Contact form
- Resume download

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Build

To create a production build:
```bash
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable components
├── pages/         # Page components
├── assets/        # Static assets
├── App.tsx        # Main App component
└── main.tsx       # Entry point
```

## Customization

1. Update the project data in `src/pages/Projects.tsx`
2. Modify the resume content in `src/pages/Resume.tsx`
3. Update contact information in `src/pages/Contact.tsx`
4. Customize the styling in `src/index.css` and `tailwind.config.js`
